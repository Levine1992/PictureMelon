### 原理逻辑
利用webview加载网页再通过Jsoup对网页进行解析得到图片或视频链接，再通过链接下载保存到手机中

功能很简单，代码也比较少，所以代码中也没做太多注释，一般都看的懂

虽然是比较简单的项目但是也希望能帮助到需要帮助的人

此项目仅供个人学习参考，切勿用于商业用途

有问题多多指教互相学习
